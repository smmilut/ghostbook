Usage and information

Usage help :

In the list of clues :

  • Click a clue to cycle between the three investigations states : unknown, clue detected, clue can't be detected
  • Click a clue state to toggle it.
  • Use the "Clear" button to clear the selection for a new mission.

In the resulting list of ghosts :

  • Click a ghost to get details about this ghost.
  • The only displayed ghosts are those who match the selected clue(s).
  • The only displayed clues are those that are not a selected clue.

Language :

Additionally, you can set a prefered list of languages from the URL parameter "lang", and they will take precedence.

Examples :

  • https://smmilut.github.io/ghostbook/?lang=fr will set language fr (French) as prefered language. For any given text, if this language is unvailable then your browser preferences will be used instead.
  • https://smmilut.github.io/ghostbook/?lang=it,fr will set language it as prefered language, and fr as secondary.

In any case, a default list of prefered languages is read from browser preferences.

If no text is available for those languages, fallback is set to English en.
