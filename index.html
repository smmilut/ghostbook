Usage

In the list of clues :

  • Click a clue to make it selected.
  • Click a selected clue to make it unselected.
  • Ctrl+click to select multiple clues.

In the resulting list of monsters :

  • Click a monster to get details about this monster.
  • The only displayed monsters are those who match the selected clue(s).
  • The only displayed clues are those that are not a selected clue.
